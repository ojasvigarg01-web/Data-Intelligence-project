# Data-Intelligence-project
“Data intelligence projects using MySQL and Power BI, featuring dynamic pricing and energy trade analytics.”

📌 1. Project Title

# Data Intelligence Projects

📌 2. Description (Intro)

This repository contains data intelligence and business analytics projects built using MySQL and Power BI. 
It includes a Dynamic Pricing and Demand Intelligence System and an Energy Trade Analytics Dashboard, 
focused on deriving actionable insights for better decision-making.

📌 3. 📂 Projects Included

1. Dynamic Pricing and Demand Intelligence System (MySQL)
- Developed using MySQL
- Focuses on demand analysis and pricing optimization
- Uses structured queries to extract business insights

 2. Energy Trade Analytics Dashboard (Power BI)
- Built using Power BI
- Interactive dashboard for energy trade analysis
- Includes KPIs, trends, and visual insights

📌 4.   Tools & Technologies

- MySQL
- Power BI
- SQL
- Data Analysis
- Data Visualization
  
📌 5. 📊 Key Features

- Data cleaning and transformation
- Business intelligence dashboards
- Demand forecasting insights
- Pricing strategy analysis
- Interactive visual reports

  📌 6. 📁 Folder Structure

- /dynamic-pricing-sql → MySQL project files
- /energy-trade-powerbi → Power BI dashboard files
  
📌 7. 🚀 How to Use

1. Clone the repository
2. Open MySQL files in any SQL environment
3. Open Power BI file (.pbix) in Power BI Desktop
4. Explore queries and dashboards

 8. 🎯 Project Objective
